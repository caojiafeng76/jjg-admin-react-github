## 1. 目标

为车间管理增加“机器设备维护”能力，支持管理员维护设备档案与设备小时费率基础参数，并自动生成：

- 单小时电费 = 功率 × 电单价 ÷ 同步工作数量
- 折旧费率 = 机器价值 ÷ 折旧年份 ÷ 年运行时长
- 设备小时费率 = 单小时电费 + 折旧费率

页面交互与“岗位基础数值设定”保持一致，采用表格 + 搜索 + 弹窗表单的 CRUD 结构。

## 2. 范围

本次交付包含以下内容：

- Supabase 新表：public.machine_equipment_maintenances
- RLS 策略：仅管理员可读写
- React 前端 feature：机器设备维护列表、搜索、创建、编辑、删除
- 路由与左侧导航接入
- 构建验证

## 3. 数据设计

### 3.1 表名

- public.machine_equipment_maintenances

### 3.2 字段说明

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| id | uuid | 主键 |
| unified_device_no | text | 统一设备编号，唯一 |
| operation | text | 工序 |
| machine_name | text | 机器名称 |
| original_no | text | 原编号，可为空 |
| power_kw | numeric(10,2) | 功率 |
| sync_work_quantity | integer | 同步工作数量 |
| electricity_unit_price | numeric(10,4) | 电单价 |
| hourly_electricity_fee | numeric(14,8) | 单小时电费，生成列 |
| machine_value | numeric(12,2) | 机器价值 |
| depreciation_years | integer | 折旧年份 |
| annual_runtime_hours | numeric(12,2) | 年运行时长 |
| depreciation_rate | numeric(14,8) | 折旧费率，生成列 |
| equipment_hourly_rate | numeric(14,8) | 设备小时费率，生成列 |
| remark | text | 备注 |
| created_at | timestamptz | 创建时间 |
| updated_at | timestamptz | 更新时间 |

### 3.3 约束与索引

- unified_device_no 唯一索引
- operation、machine_name、unified_device_no 非空校验
- power_kw、electricity_unit_price、machine_value 非负校验
- sync_work_quantity、depreciation_years、annual_runtime_hours 必须大于 0
- updated_at 使用统一触发器自动维护

## 4. 权限设计

- 开启 RLS
- authenticated 用户仅在 is_admin() 为 true 时允许增删改查
- 当前版本不向员工侧和班组长侧开放

## 5. 前端设计

### 5.1 页面位置

- 一级分组：车间管理
- 菜单名称：机器设备维护
- 路由：/machine-equipment-maintenance

### 5.2 页面结构

页面沿用岗位基础数值设定模式：

1. 顶部工具栏：新增、编辑、删除
2. 搜索区：统一设备编号、工序、机器名称
3. 表格区：展示设备基础信息与自动计算费率
4. 弹窗表单：维护字段输入，底部展示自动计算预览卡片

- 管理员可进入“机器设备维护”页面
