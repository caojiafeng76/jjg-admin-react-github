# 岗位基础数值设定开发文档

## 1. 目标

为车间管理增加“岗位基础数值设定”能力，支持管理员维护各工种的基础核算参数，并自动生成：

- 工时费 = 标准收入 / 月标准工作时间
- 月标准工作时间 = 每日工作时间 × 工作天数

该功能面向管理员使用，页面风格与“订单管理”保持一致，采用表格 + 搜索 + 弹窗表单的 CRUD 交互。

## 2. 范围

本次交付包含以下内容：

- Supabase 新表：`public.job_base_settings`
- RLS 策略：仅管理员可读写
- React 前端 feature：岗位基础数值设定列表、搜索、创建、编辑、删除
- 新增管理路由与左侧导航入口
- 开发文档与验收 checklist

## 3. 数据设计

### 3.1 物理表命名

虽然业务名称为“岗位基础数值”，但数据库物理表名采用 `job_base_settings`，原因如下：

- 与当前仓库全部英文 snake_case 表命名风格保持一致
- 避免后续 Supabase 类型生成、前端 service 引用、SQL 维护出现中文标识符兼容问题
- 中文名称通过表注释和前端 UI 完整保留

### 3.2 表结构

表名：`public.job_base_settings`

字段说明：

| 字段名                   | 类型            | 说明                   | 是否可编辑 |
| ------------------------ | --------------- | ---------------------- | ---------- |
| `id`                     | `uuid`          | 主键                   | 否         |
| `job_name`               | `text`          | 工种                   | 是         |
| `standard_income`        | `numeric(12,2)` | 标准收入               | 是         |
| `daily_work_hours`       | `numeric(6,2)`  | 每日工作时间           | 是         |
| `working_days`           | `integer`       | 工作天数               | 是         |
| `monthly_standard_hours` | `numeric(10,2)` | 月标准工作时间，生成列 | 否         |
| `hourly_fee`             | `numeric(14,8)` | 工时费，生成列         | 否         |
| `created_at`             | `timestamptz`   | 创建时间               | 否         |
| `updated_at`             | `timestamptz`   | 更新时间               | 否         |

### 3.3 约束与规则

- `job_name` 非空且唯一
- `standard_income >= 0`
- `daily_work_hours > 0`
- `working_days > 0`
- `updated_at` 通过 `public.update_updated_at_column()` 触发器自动维护

### 3.4 计算规则

$$
月标准工作时间 = 每日工作时间 \times 工作天数
$$

$$
工时费 = \frac{标准收入}{月标准工作时间}
$$

说明：

- 两个结果字段均放在数据库生成列中，避免前后端重复计算导致口径不一致
- 前端表单提供实时预览，但最终以数据库结果为准

## 4. 权限设计

- 路由权限：仅 `admin`
- 数据权限：Supabase RLS 仅允许管理员对 `job_base_settings` 执行 `select/insert/update/delete`
- 员工端、班组长端不展示入口，不开放访问

## 5. 前端设计

### 5.1 页面位置

- 一级分组：车间管理
- 菜单名称：岗位基础数值设定
- 路由：`/job-base-setting`

### 5.2 页面结构

参考“订单管理”页面，结构如下：

1. 工具栏：添加 / 编辑 / 删除
2. 搜索栏：按工种搜索
3. 数据表格：分页显示岗位基础数值
4. 弹窗表单：创建和编辑

### 5.3 表单交互

可编辑字段：

- 工种
- 标准收入
- 每日工作时间
- 工作天数

只读预览字段：

- 月标准工作时间
- 工时费

## 7. 验收标准

- 管理员可进入“岗位基础数值设定”页面
- 可创建、编辑、删除岗位基础数值
- 工时费和月标准工作时间自动计算且展示正确
- 同名工种不可重复创建
- 非管理员无法通过路由或 API 访问该资源
- 分页、搜索、弹窗表单交互正常
